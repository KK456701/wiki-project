# 指标结果差异诊断

> **结论速览**
> 用一到两句话说明差异是否已解释、停止在哪一层，以及最重要的确认结论。

## 双方结果

用并列表格展示当前生效口径结果与用户或文件结果，包括分子、分母、指标率和统计区间。

## 候选口径试算

用表格展示每个已执行候选的分子、分母、指标率、匹配等级和原因判断；同时说明仍未匹配的维度。

## 诊断结论

展示结论代码、停止层级、已解释记录数和已确认原因。只有 cause_confirmed=true 才能使用“已确认”。

## 证据限制

说明尚未解释的差异、缺少的外部证据或逐条记录限制，不得断言用户结果错误。

不要输出患者明细、SQL 正文、占位符和本模板说明。
