# 急会诊及时到位率_本院口径

## 本院指标定义

急会诊请求发出后，25分钟内到达现场的急会诊次数占同期急会诊总次数的比例。

## 本院计算公式

急会诊及时到位率 = (急会诊记录中 25分钟内到位的急会诊次数 / 同期急会诊总次数) × 100%
